# Interactive Website

A fun website that works on phones and computers! It has a quiz, photo slideshow, and weather info.

## What You Need
Just 3 files in one folder:
- `index.html` (the website)
- `styles.css` (makes it pretty)
- `script.js` (makes it interactive)
